# vMall

基于Laravel和Vue的无线商城

重构自[iMall](https://github.com/PassionZale/iMall)

## install

### Be install

```bash

composer install

composer artisan key:generate

composer artisan migrate

```

### Fe install

``` bash

yarn install

```

## dev

```bash

php artisan serve

yarn dev

```
